# Songpress 1.1 released!

We are proud to release Songpress - Il Canzonatore 1.1.

New features in this release include:

- Song transposition with key autodetection
- Support of several chord notations (American/English, Italian/Spanish, German, French, Portuguese); automatic notation conversion
- Quick editing of chords
- Exporting formatted songs as PNG images
- Some bugfixes and improvements:
    - Removed flicker in preview pane while editing song
    - It is possible to insert comments in the title block
    - Improved drag'n'drop
