# Songpress 1.5 rilasciato!

Songpress 1.5 introduce nuove funzionalità richieste da molti utenti:

- È possibile etichettare strofe e ritornelli con etichette personalizzate, come "Bridge", "Strumentale", ecc.
- È possibile impaginare le parti strumentali (ovvero, solo accordi)
- Estensione predefinita personalizzabile per i file chordpro
