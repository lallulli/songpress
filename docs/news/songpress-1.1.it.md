# Rilasciata la versione 1.1 di Songpress!

Siamo orgolgiosi di rilasciare la versione 1.1 di Songpress - Il Canzonatore.

Di seguito, le nuove funzionalità:

- Trasposizione della canzone con l'autodetect degli accordi
- Supporto per diverse notazioni degli accordi (Americana/Inglese, Italiana/Spagnola, Tedesca, Francese, Portoghese); conversione automatica tra le diverse notazioni.
- Editing rapido degli accordi
- Esportazione in immagine PNG delle canzoni formattate
- Alcune correzioni di bug e miglioramenti:
    - Rimosso lo sfarfallio nel riquadro di anteprima durante la modifica del brano.
    - È possibile inserire commenti nel blocco del titolo
    - Miglioramento del drag'n'drop
