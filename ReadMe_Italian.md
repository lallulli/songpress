### Importante per l'installazione di wxpython su Ubuntu / Linux Mint

Su Ubuntu / Mint, pip non può installare wxpython di default.

Può essere installato usando questo comando:

```pip3 install -U -f https://extras.wxpython.org/wxPython4/extras/linux/gtk3/ubuntu-20.04 wxPython```

### Songpress

Songpress è un programma per Windows e Linux in grado di creare e modificare file Chord Pro.

### Songpress ### Per iniziare

Apri un terminale nella cartella src, quindi

```python3 main.py```

### Caratteristiche

- Crea spartiti per chitarra di alta qualità (testi e accordi)

- Facile da imparare, veloce da usare

- Puoi inserire brani formattati in qualsiasi applicazione Windows per creare il tuo songbook con la massima flessibilità (Microsoft Word, LibreOffice, Microsoft Publisher, Inkscape, ecc.).

- Esporta brani formattati in PNG e HTML (pagine web e frammenti).

- Trasposizione degli accordi con rilevamento automatico della tonalità

- Semplificazione degli accordi per principianti: determina la tonalità più facile per suonare il tuo brano e trasponila automaticamente

- Supporta diverse notazioni di accordi: americana (C, D, E), italiana (Do, Re, Mi), francese, tedesca e portoghese; Conversione della notazione

- Supporta i formati di accordi Chordpro ​​e Tab (ovvero, accordi a due righe)

- Rimuove le righe vuote errate (ad esempio, brani copiati e incollati da siti web) e la notazione degli accordi incoerente
